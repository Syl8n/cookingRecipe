# Cooking Recipe

레시피를 공유하며 간단한 소통도 할 수 있는 커뮤니티 사이트입니다.

점점 시간이 지남에 따라 식생활 등의 소소한 삶의 질을 챙기는 사람이 늘어났습니다.

이에 공공 레시피 데이터와 각자가 만들어 낸 커스텀 레시피를 공유하고, 대화하며, 평가도 하는 사이트를 제작해보게 되었습니다.
